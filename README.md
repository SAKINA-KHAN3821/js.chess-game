# ♟️ JavaScript Chess Game

A fully functional chess game built using **HTML, CSS, and JavaScript**, featuring real-time move validation, piece logic, and an interactive UI.

---

## 🚀 Features

- ♟️ Complete 8x8 chess board
- 🔄 Turn-based gameplay (White vs Black)
- 🎯 Highlight valid moves for each piece
- ⚔️ Capture opponent pieces
- 👑 Special moves (Castling)
- 🎨 Smooth animations and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- jQuery
